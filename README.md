# HardCopy Pro

## Overview

HardCopy Pro is a screen capture tool designed for Windows users to quickly and efficiently take screenshots of the entire desktop, active windows, or selected areas. Its purpose is to provide a straightforward utility for capturing, editing, and saving screen content without unnecessary complexity.

## Key Features

**Capture full screen, active window, or custom regions**  
**Support for multiple image formats including PNG, JPEG, BMP, GIF, and TIFF**  
**Simple editing tools for annotations, cropping, and resizing**  
**Automatic file naming with date and time stamps**  
**Configurable hotkeys for streamlined workflow**  
**Direct saving or copying to clipboard for quick use**  
**Lightweight performance with minimal system resource usage**
